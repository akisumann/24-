# 七年巫女神

国と神の契約により、常時50人の登録巫女を維持し、7年ごとの大儀と世代推移を観測するブラウザゲームです。

## 遊び方

`index.html` を通常のWebブラウザで開いてください。GitHub Pagesでは、公開URLからiPhoneでも遊べます。

公開先：`https://akisumann.github.io/24-/`

## 設定資料

ゲーム上の処理と、世界の中での意味を混同しないよう、資料を分けています。

- [`GAME_RULES.md`](GAME_RULES.md) — 年齢進行、能力計算、選抜、役務、親類縁者などのゲーム仕様
- [`LEVEL_GUIDE.md`](LEVEL_GUIDE.md) — 世界共通の1〜100レベル指標と物語上の強さの目安
- [`WORLD_SETTING.md`](WORLD_SETTING.md) — 巫女の生活、仕事、政治、婚姻、国家制度としての意味

巫女団の役務はゲーム上の適性分類であり、巫女本人の日常の本業を固定するものではありません。
巫女は7年ごとの大儀に備えつつ、普段は別の仕事、政治、家庭生活、穏やかな暮らしなどを自由に送ります。

## 画面と機能（v16で追加）

- **タイル型ダッシュボードUI** — 上部に固定HUD（経過年・神レベル・現役・親類縁者・国家評判）と「七年進める／×10」進行ボタン、「全開く／全畳む」を配置。本文は各パネルをタイル状に並べ、各枠は右上のボタンで折りたたみできる。
- **推移グラフ** — 神レベル・国家評判・親類縁者・成人平均レベルの長期推移をスパークラインで表示。
- **国家評判の動的化** — 経過年による無制限の上昇をやめ、現在の実力に基づく均衡値へ減衰しつつ、神羅巫女の実績で加点する（上下する）値に変更。
- **役務の上限撤廃と時代の需要** — 役務の定員を撤廃し、各巫女を第一適性へ配属。役務の顔ぶれから、数が上位2つを合わせた「〇〇と〇〇の時代」を判定し、時代の移り変わりを年表として記録する。
- **巫女の活躍の物語** — 巫女を選ぶと、役務・スキル・立場・一族・その時代の需要から「国での活躍」を物語文で生成して表示する。
