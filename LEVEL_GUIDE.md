# 世界共通レベル指標

この文書は、人物・冒険者・兵士・巫女・神などのレベルを物語上で評価する際に用いる、1〜100の世界共通指標である。

ゲーム内の数値処理は [`GAME_RULES.md`](GAME_RULES.md) を参照し、人物の強さや社会的な位置づけを描写する際は本指標を優先する。

| レベル | 目安 |
|---|---|
| Lv1〜10 | 一般人・駆け出し |
| Lv11〜20 | 下級冒険者・一般兵士 |
| Lv21〜30 | 1人前の冒険者・熟練兵 |
| Lv31〜40 | ベテラン・地方の実力者 |
| Lv41〜50 | 地方上位・精鋭クラス |
| Lv51〜60 | 国家上位・騎士団長・宮廷魔導士・高ランク冒険者 |
| Lv61〜70 | 英雄級・国家の切り札 |
| Lv71〜80 | 大陸でも名の知れた超一流 |
| Lv81〜90 | 伝説級 |
| Lv91〜99 | 神話級 |
| Lv100 | 理論上の最高到達点 |

## 運用上の注意

- 同じレベル帯でも、ステータス配分、技能、装備、相性、経験によって実際の勝敗や得意分野は異なる。
- レベルは総合的な能力水準の目安であり、特定分野だけの突出をそのまま表すものではない。
- Lv71以上は単なる国内上位ではなく、大陸規模で名が知られうる領域として扱う。
- Lv81以上は通常の超一流を越え、歴史や伝承に残る存在として扱う。
- Lv100は通常の育成で容易に到達する値ではなく、この世界における理論上の最高到達点である。
