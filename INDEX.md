# 設定資料インデックス（AI向け文書地図）

このリポジトリの設定資料の入口。**どこに何が書いてあるか**を辿るための地図。
AIがこの世界の物語を書く・ゲームを扱うときは、まずここで担当ファイルを特定してから、その本文を読む。

---

## 0. まず読むもの（順番）

1. **`AI_HANDOFF.md`** — 最優先。①物語を書く前の**完全セーブ直読ゲート**（推測・捏造の禁止）、②**数値の書き方**（算用数字／神レベル≤100など）、③**外見描写ルール**、④実行規約、⑤ファイル構成と拡張設計。
2. **このINDEX** — 話題から担当ファイルを引く。
3. 各担当ファイル本文。

実データの取得方法（本物のコードを走らせて読む）は `AI_HANDOFF.md` と `ai-policy.js`、ヘッドレス実行版 `miko-headless.js` を参照。**再実装・移植・推測は禁止**。

## 1. 文書の優先関係

同じ事柄が複数文書に触れられている場合の優先順位。

| 種類 | 優先する文書 |

| 数値・抽選・年齢進行・退任・評判・画面表示 | `GAME_RULES.md`（数値仕様）／`RETIREMENT_RULE.md`（退任の詳細） |

| 大儀の場面・降臨儀礼・大寵愛 | `SHINSHODEN.md` |
| 巫女服の**構造仕様**（形・寸法） | `MIKO_FORMAL_WEAR.md`（正装＝御種衣の正典） |

数値と世界観の切り分け原則：**「実際に動く数値」は `GAME_RULES.md`、「世界の中での意味」は `WORLD_SETTING.md`**。

## 4. 主要用語（定義の在り処）

- **御種衣（おたねぎぬ）** — 一枚の中央布からなる生殖用巫女装束。構造は `MIKO_FORMAL_WEAR.md`、機能は `MIKO_CLOTHING_AND_MARRIAGE.md`。「神聖衣／基本正装／中央布／前垂れ」も同じものの呼び分け。
- **淫欲神（いんよくしん）** — この国が契約する神。数ある神々（豊穣神・武勇神・財宝神・叡智神など）のうち、子孫繁栄・血統に特化した系統。→ `WORLD_SETTING.md`「神という存在」。
- **大儀（たいぎ）** — 七年に一度、上位10人が神と子作りする国家事業。場は神床殿。→ `WORLD_SETTING.md`「神の顕現と大儀の対話」／`SHINSHODEN.md`。
- **神床殿（しんしょうでん）** — 大儀を行う場。降臨儀礼・自己紹介・対話・子作り。→ `SHINSHODEN.md`。
- **大寵愛（だいちょうあい）** — 十人全員を抱いたうえで最も激しく求めた一人へ注がれる最も深い寵愛（独占ではない）。その母の娘は能力乱数の下限が上がる。→ `SHINSHODEN.md`「大寵愛」。
- **神羅巫女（しんらみこ）** — 神レベルを上昇させた（神を成長させた）巫女の称号。極めて稀。→ `GAME_RULES.md`／`WORLD_SETTING.md`／`MIKO_TURNOVER_SETTING.md`。
- **加護** — 神の力による確実な受胎と、母子を守り体を整える力。→ `WORLD_SETTING.md`「神の加護と確実な受胎」。
- **潜在レベル** — 最大能力7項目の平均。退任・大儀選抜の基準。→ `GAME_RULES.md`。

## 5. 表現・境界の基本（詳細は AI_HANDOFF.md）

- 数値は**算用数字**で書く。神レベルは**100が上限**。評判・親類縁者などの範囲は `AI_HANDOFF.md` を厳守。
- 性的・露出に関わる描写は**成人した巫女（大儀に臨む二十歳〜三十四歳）**が対象。名簿の**六歳・十三歳は国家育成対象**で、年齢相応の教育を受けて育つ段階として描き、性的な行為・描写の対象にはしない。→ `WORLD_SETTING.md`「巫女の精神性と性への構え」。
